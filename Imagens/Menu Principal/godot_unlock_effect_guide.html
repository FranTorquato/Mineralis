1
Criar o shader de cinza
O coração do efeito bloqueado
res://shaders/gray_card.gdshader

Crie a pasta shaders/ no seu projeto e salve este arquivo:

shader_type canvas_item; // 0.0 = colorido 1.0 = cinza total uniform float gray_amount : hint_range(0.0, 1.0) = 1.0; // 1.0 = normal 0.6 = escurecido uniform float brightness : hint_range(0.5, 1.5) = 0.6; void fragment() { vec4 color = texture(TEXTURE, UV); float gray = dot(color.rgb, vec3(0.299, 0.587, 0.114)); vec3 result = mix(color.rgb, vec3(gray), gray_amount); result *= brightness; COLOR = vec4(result, color.a); }
O uniform gray_amount é o que vamos animar: vai de 1.0 (cinza) para 0.0 (colorido) durante o desbloqueio.
2
Montar a cena do card
Estrutura de nós no Godot
Scene > New Scene

Crie card_phase.tscn com esta hierarquia exata:

📄 Control ← card_phase.gd aqui ├── TextureButton "CardButton" (tamanho: 160×160) │ ├── TextureRect "CardImage" (Expand Mode: Ignore Size) │ └── TextureRect "LockOverlay" (ícone cadeado, centralizado) ├── ColorRect "FlashOverlay" (160×160, color branco, visible=false) └── AudioStreamPlayer "UnlockSound"
Selecione CardImage → Inspector → CanvasItem → Material → New ShaderMaterial → arraste o gray_card.gdshader no campo Shader.

No Control raiz, defina Custom Minimum Size para 160 × 160 e Layout > Anchors como "Full Rect" para que caiba no mapa.

3
Escrever card_phase.gd
Script completo com estados e animação
res://scripts/card_phase.gd
extends Control # Arraste os nós no Inspector @export var phase_id : String @export var card_texture : Texture2D @onready var card_btn = $CardButton @onready var card_img = $CardButton/CardImage @onready var lock_icon = $CardButton/LockOverlay @onready var flash = $FlashOverlay @onready var sfx = $UnlockSound var mat : ShaderMaterial var unlocked := false signal card_pressed(id) func _ready(): card_img.texture = card_texture mat = card_img.material as ShaderMaterial card_btn.pressed.connect(_on_pressed) _apply_locked(false) # estado inicial sem animação func unlock(animate := true): if unlocked: return unlocked = true if animate: _play_unlock() else: _apply_unlocked(false) func _apply_locked(anim): card_btn.disabled = true lock_icon.visible = true mat.set_shader_parameter("gray_amount", 1.0) mat.set_shader_parameter("brightness", 0.6) func _apply_unlocked(anim): card_btn.disabled = false lock_icon.visible = false mat.set_shader_parameter("gray_amount", 0.0) mat.set_shader_parameter("brightness", 1.0) func _on_pressed(): if unlocked: emit_signal("card_pressed", phase_id)
4
Animar o desbloqueio
A função _play_unlock() em detalhe

Adicione esta função no mesmo card_phase.gd. O efeito tem 4 etapas em sequência:

🔒
bloqueado
func _play_unlock(): sfx.play() # ── Etapa 1: flash branco ────────────────────── flash.color = Color(1, 1, 1, 0.85) flash.visible = true var t = create_tween() t.set_parallel(true) t.tween_property(flash, "color:a", 0.0, 0.25) # ── Etapa 2: pulse de escala ─────────────────── t.tween_property(self, "scale", Vector2(1.18, 1.18), 0.12) t.set_parallel(false) t.tween_property(self, "scale", Vector2(1.0, 1.0), 0.22)\ .set_trans(Tween.TRANS_BACK).set_ease(Tween.EASE_OUT) # ── Etapa 3: desgrayificar (shader) ─────────── t.set_parallel(true) t.tween_method( func(v): mat.set_shader_parameter("gray_amount", v), 1.0, 0.0, 0.5) t.tween_method( func(v): mat.set_shader_parameter("brightness", v), 0.6, 1.0, 0.5) # ── Etapa 4: esconder cadeado + shimmer ─────── t.set_parallel(false) t.tween_callback(func(): lock_icon.visible = false card_btn.disabled = false flash.visible = false _play_shimmer() )
5
Efeito shimmer (brilho diagonal)
O toque final após o desbloqueio

Adicione também no card_phase.gd. Cria um ColorRect temporário que atravessa o card:

func _play_shimmer(): var s = ColorRect.new() s.color = Color(1, 1, 1, 0.35) s.size = Vector2(50, 160) s.position = Vector2(-55, 0) s.rotation = deg_to_rad(20) # diagonal add_child(s) var t = create_tween() t.tween_property(s, "position:x", 180.0, 0.4)\ .set_trans(Tween.TRANS_SINE).set_ease(Tween.EASE_IN_OUT) t.tween_callback(s.queue_free) # Hover effect (só quando desbloqueado) func _on_mouse_entered(): if not unlocked: return create_tween().tween_property(self,"scale",Vector2(1.07,1.07),0.1) func _on_mouse_exited(): if not unlocked: return create_tween().tween_property(self,"scale",Vector2(1.0,1.0),0.15)
Conecte os sinais mouse_entered e mouse_exited do CardButton nas funções acima. No Inspector do CardButton → Node → Signals.
6
Configurar no Inspector
Cada card precisa de 3 ajustes
Selecione o nó raiz Control de cada card
  • Em Script Variables → Phase Id: preencha com "1_1", "1_2", "1_3" etc.
  • Em Script Variables → Card Texture: arraste o SVG/PNG do card correspondente.
  • Selecione CardImage → Material → confirme que o ShaderMaterial com gray_card.gdshader está aplicado.
  • Selecione LockOverlay → arraste uma textura de cadeado (ou deixe um TextureRect vazio que só mostra cor de sobreposição).
  • Selecione FlashOverlay (ColorRect) → defina Color como branco e Visible = false no começo.
Para usar o mesmo ShaderMaterial em vários cards sem que compartilhem estado, no Inspector do Material clique no ícone de corrente e escolha "Make Unique" em cada instância.
7
Chamar o unlock() do mapa
Conectando tudo no world_map.gd
res://scripts/world_map.gd
extends Control @onready var card_1_1 = $Cards/Card_1_1 @onready var card_1_2 = $Cards/Card_1_2 @onready var card_1_3 = $Cards/Card_1_3 func _ready(): _setup_cards() ProgressManager.phase_unlocked.connect(_on_unlocked) func _setup_cards(): # Aplica estado salvo SEM animação var cards = {"1_1":card_1_1, "1_2":card_1_2, "1_3":card_1_3} for id in cards: if ProgressManager.is_unlocked(id): cards[id].unlock(false) # false = sem anim cards[id].card_pressed.connect(_on_card_pressed) func _on_unlocked(id): # Chamado pelo ProgressManager — COM animação match id: "1_1": card_1_1.unlock(true) "1_2": card_1_2.unlock(true) "1_3": card_1_3.unlock(true) func _on_card_pressed(id): # Navega para a fase get_tree().change_scene_to_file("res://scenes/phase_%s.tscn" % id) # Chame isto ao FINAL de cada fase jogada: # ProgressManager.complete_phase("1_1")
O fluxo completo: jogador termina fase → chama ProgressManager.complete_phase("1_1") → o manager desbloqueia "1_2" → emite o sinal phase_unlocked → o mapa recebe e chama card_1_2.unlock(true) → animação toca.
